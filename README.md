# 🐾 Shopet

### Modern Pet Shop built with Next.js

A modern, responsive and user-friendly pet shop website built with Next.js, featuring product pages, blog, authentication UI, FAQ, contact page, order tracking and more.

# 📦 Installation

```bash
git clone https://github.com/yourusername/shopet.git

cd shopet

npm install

npm run dev
```

---

# 📚 Packages

```bash
npx create-next-app@latest shopet
```

### Form

```bash
npm install react-hook-form
```

### Validation

```bash
npm install yup
npm install @hookform/resolvers
```

### Notifications

```bash
npm install react-hot-toast
```

### Server State

```bash
npm install @tanstack/react-query
```

### Icons

```bash
npm install lucide-react
```
